# Vulnerable Packages Knowledge Base

Package Name: Newtonsoft.Json
Vulnerable Versions: < 13.0.1
High-Severity Vulnerability: Improper handling of highly nested JSON arrays and objects can lead to a Denial of Service.
Recommendation: Update to a secure version.
Safe Version: 13.0.3

Package Name: AutoMapper
Vulnerable Versions: < 7.0.0
Medium-Severity Vulnerability: Potential information disclosure in specific edge cases.
Recommendation: Update to a supported version.
Safe Version: 13.0.1
